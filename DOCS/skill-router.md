## VIRON-spezifische Regeln

1. **aso** wird NIEMALS genutzt (keine Mobile App)
2. **paywalls** wird NIEMALS genutzt (VIRON ist Agentur, kein SaaS)
3. **churn-prevention** wird NIEMALS genutzt (Projekt-basiert, keine Subscriptions)
4. **community-marketing** wird ERST genutzt wenn Community existiert
5. **co-marketing** wird ERST genutzt wenn Partnerships anstehen
6. **referrals** wird ERST genutzt wenn Kundenstamm vorhanden
7. **free-tools** wird priorisiert für Lead-Gen (ROI Calculator, AI Audit Tool)
8. **directory-submissions** wird bei JEDEM Launch genutzt
9. **ai-seo** hat Priorität vor **seo-audit** für DACH-KMU Zielgruppe
10. **cold-email** ist PRIMARY Outreach-Kanal für B2B Akquise
